# 一建经济刷题系统

2026年一级建造师《建设工程经济》刷题系统，纯前端静态实现，无需服务器。

## 功能特性

- **章节选择首页**：20章目录，清晰展示进度
- **一页一题**：专注答题，避免干扰
- **实时计分**：单选对1分，多选按规则计分
- **断点续做**：localStorage 自动保存进度
- **错题重做**：一键重做错题，支持选择性重练
- **分类筛选**：全部/基础题/进阶题/未答/错题
- **响应式设计**：适配桌面和移动端

## 使用方法

1. 直接打开 `index.html` 即可使用
2. 或部署到任意静态服务器（GitHub Pages、Vercel 等）

## 项目结构

```
一建经济刷题/
├── index.html          # 章节选择首页
├── ch01.html           # 第1章刷题页（已上线）
├── README.md           # 说明文档
├── .gitignore          # Git忽略规则
└── chapters/           # 章节源文件
    ├── ch01.md         # 第1章题目
    └── ch01-ch20.txt   # 教材提取文本
```

## 计分规则

- **单选题**：选对得 1 分，选错得 0 分
- **多选题**：选对一个得 0.5 分，全对得 2 分，错选得 0 分

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无任何外部依赖
- localStorage 本地存储
